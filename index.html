TITLE

A tag de Title é uma tag de código HTML que permite atribuir um título a uma página da web. Este título pode ser encontrado na barra de título do navegador, bem como nas páginas de resultados dos mecanismos de pesquisa (SERP). É fundamental adicionar e otimizar as tags de título do seu site, pois elas desempenham um papel essencial em termos de classificação orgânica (SEO).

H1-6

Os elementos HTML h1–h6 representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.
